# 什么是Agent

Agent（智能体）是一种能够代表你执行操作的AI助手。

## 与对话式AI的区别

与单纯回答问题的对话式AI不同，Agent可以：
- 读写本地文件
- 执行终端命令
- 搜索网页信息
- 访问代码仓库
- 自动化重复任务

## 核心能力

1. **理解任务**：理解自然语言描述的任务
2. **调度工具**：调用各种工具完成任务
3. **执行操作**：在系统中执行实际操作

## 常用Agent工具

- Claude Code
- Cursor
- Windsurf
- GitHub Copilot
- OpenCode
