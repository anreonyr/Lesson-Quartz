# 库的使用流程

## 1. 安装库

使用包管理工具安装：

```bash
# Python
pip install 库名

# Node.js
npm install 包名
```

让AI帮你：
> 告诉我如何安装Python的requests库

## 2. 导入库

```python
import 库名
from 库名 import 具体功能
```

让AI帮你：
> 在Python中，如何导入并使用PIL库来处理图片？

## 3. 查看文档

让AI帮你：
> requests库的官方文档在哪里？如何查看某个函数的具体用法？

## 4. 调试使用

遇到错误时，让AI帮你：
> 我在使用requests库时遇到了ConnectionError错误，请帮我分析原因
