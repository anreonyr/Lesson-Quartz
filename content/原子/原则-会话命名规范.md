# 会话命名规范

给会话起个好名字，方便以后查找。

## 好名字的特点

- 清晰表达会话内容
- 便于快速识别
- 包含项目/主题关键字

## 命名示例

❌ 不好：
```
新会话 1
对话
未命名
```

✅ 好：
```
Todo项目-数据存储模块
Bug修复-登录问题
学习-Python装饰器
电商项目-用户模块
```

## 命名规范建议

1. **项目名-功能/主题**：如 "Todo项目-数据存储"
2. **类型-描述**：如 "Bug修复-支付问题"
3. **学习-主题**：如 "学习-正则表达式"

## 好处

- 快速找到历史会话
- 便于会话管理
- 避免重复提问
