# Prompt进阶技巧

## 使用分隔符

使用代码块或特殊符号分隔不同部分：

```
以下是需求：
1. ...
2. ...

以下是参考代码：
```
code here
```
```

## 给出示例

如果有类似例子，给AI参考：

> 按照以下格式输出JSON：
```json
{"name": "张三", "age": 20}
```

## 让AI解释自己

让AI解释生成的代码，帮助你理解：

> 请写出代码后，解释每行代码的作用

## 分步引导

对于复杂任务，分步引导：

> 第一步：帮我设计数据库结构
> 第二步：帮我写用户注册接口
> 第三步：帮我写登录接口

## 指定角色

给AI一个特定角色：

> 你是一个资深Python工程师，请帮我审查这段代码...

## 提供背景信息

越多相关背景，AI理解越准确：

> 这是一个Django项目，使用SQLite数据库，目标是...
