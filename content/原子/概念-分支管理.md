# 分支管理

分支是独立的开发线，可以在分支上做实验，不影响主代码。

## 基本操作

### 创建分支

```bash
git checkout -b 新分支名
```

### 切换分支

```bash
git checkout 分支名
```

### 合并分支

```bash
git merge 分支名
```

## 使用分支做实验

```bash
# 创建新分支
git checkout -b ai-experiment

# 在分支上让AI改代码

# 如果改坏了，直接删除分支
git checkout main
git branch -D ai-experiment

# 如果改好了，合并到主分支
git merge ai-experiment
```

## 分支命名规范

- `main`：主分支
- `dev`：开发分支
- `feature/功能名`：新功能
- `bugfix/问题描述`：Bug修复
- `ai-experiment`：AI实验

## 好处

- 可以在不影响主线的情况下进行实验
- 便于代码审查
- 支持并行开发
