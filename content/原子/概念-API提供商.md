# API提供商

API提供商是提供底层AI模型能力的公司或服务。

## 主要职责

- 提供底层AI模型能力
- 维护和更新模型
- 提供API接口供Agent工具调用

## 常见API提供商

| 提供商 | 代表模型 |
|--------|----------|
| MiniMax | MiniMax系列 |
| GLM | 智谱清言 |
| DeepSeek | DeepSeek系列 |
| Kimi | 月之暗面 |

## 选择考量因素

- 模型的语言理解能力
- 代码生成能力
- 价格
- 响应速度和稳定性

> 最重要的因素：**价格**
