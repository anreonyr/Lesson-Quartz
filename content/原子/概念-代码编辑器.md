# 代码编辑器

代码编辑器是编写和编辑代码的用户界面。

## 常见代码编辑器

| 编辑器 | 特点 |
|--------|------|
| VS Code | 功能丰富，插件生态强大 |
| JetBrains | 专业级，功能全面 |
| Zed | 高性能，Rust编写 |
| Sublime Text | 轻量，快速 |

## 选择考量

- 是否与熟悉的Agent工具集成
- 是否需要完整的IDE功能
- 性能和资源占用
- 跨平台支持

## 推荐组合

- OpenCode + Zed
- Claude Code / Cursor (自带编辑器)
