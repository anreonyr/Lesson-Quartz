# Prompt结构

一个结构化的Prompt通常包含以下部分。

## 结构组成

1. **角色设定（可选）**
   - 设定AI扮演的角色
   - 如"你是一个Python程序员"

2. **任务描述**
   - 告诉AI你要做什么
   - 如"写一个函数"

3. **具体要求**
   - 输入输出格式
   - 约束条件
   - 边界情况处理

4. **格式要求**
   - 代码风格
   - 输出格式
   - 返回形式

## 示例

```
[角色] 你是一个Python程序员
[任务] 请写一个函数，计算列表中的平均值
[要求]
  - 输入是一个数字列表
  - 返回平均值
  - 如果列表为空，返回None
  - 使用类型注解
[格式] 用Python实现，包含docstring
```
