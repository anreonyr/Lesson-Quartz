# 代码测试

## 为什么测试用例如此重要

1. **AI的代码不一定对**：AI可能会"自信地出错"
2. **手动测试容易遗漏**：边界情况、异常情况
3. **测试用例可以重复运行**：以后修改代码时也能快速验证
4. **让AI帮你生成测试**：你可以要求AI生成测试用例

## 用pytest编写简单测试

```python
# 文件名以 test_ 开头
# 测试函数以 test_ 开头

def test_add():
    """测试添加待办功能"""
    manager = TodoManager('test_todos.json')
    manager.add("测试待办")
    assert len(manager.todos) == 1
    assert manager.todos[0]['content'] == "测试待办"

def test_done():
    """测试完成待办功能"""
    manager = TodoManager('test_todos.json')
    manager.add("测试待办")
    manager.done(1)
    assert manager.todos[0]['completed'] == True
```

运行测试：
```bash
pytest test_example.py -v
```
