# 会话模板

## 会话开头模板

每次新建会话时，可以这样开头：

```
项目：Todo待办事项应用
技术栈：Python + SQLite + Flask
项目结构：
  app.py          # 主程序
  models.py       # 数据模型
  routes.py       # 路由
  utils.py        # 工具函数
当前任务：实现用户认证功能
```

## 项目结构说明模板

每个新项目开始时，先告诉AI项目结构：

```
这是一个Todo项目，结构如下：
- main.py 主程序
- models.py 数据模型
- storage.py 存储模块
请根据这个结构帮我写代码
```

## 代码引用模板

让AI参考之前的代码时，明确指出：

```
在之前写的 storage.py 里的 Storage 类基础上，添加一个新方法...
```

或者直接贴上相关代码。

## 会话总结模板

让AI定期生成会话总结：

```
请总结一下我们这个会话完成了什么，代码结构是怎样的
```
