# 单例模式

**意图**：确保一个类只有一个实例，并提供全局访问点。

## 解决的问题

- **接口简化**：全局只有一个实例，调用方不需要关心创建和传递
- **解耦**：调用方不需要知道其他代码也在用同一个实例

## 适用场景

- 数据库连接
- 配置对象
- 日志记录器

## 生活例子

就像只有一个校长（或只有一个班级群群主），不需要重复创建。

## 代码示例

```python
class Config:
    _instance = None

    def __new__(cls):
        if cls._instance is None:
            cls._instance = super().__new__(cls)
            cls._instance.settings = {}
        return cls._instance

# 使用 - 调用方不需要知道还有谁在用这个实例
config1 = Config()
config2 = Config()
config1.settings['theme'] = 'dark'
print(config2.settings['theme'])  # 输出: dark
print(config1 is config2)  # 输出: True
```
