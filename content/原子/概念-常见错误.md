# 常见错误

## 1. 脱裤子放屁

```python
# AI写的
def get_status(is_active):
    if is_active == True:
        return True
    else:
        return False

# 应该直接是
def get_status(is_active):
    return is_active
```

## 2. 画蛇添足

```python
# AI加了不必要的复杂
class Database:
    def __init__(self):
        self.connection = None
        self.config = load_config()

    def connect(self):
        if self.connection is None:
            self.connection = create_connection()
        return self.connection

# 其实可能只需要
conn = sqlite3.connect('db.sqlite')
```

## 3. 过度设计

AI设计了一套复杂的架构，其实你只需要一个简单的函数。

## 4. 复制粘贴

有时候AI会从训练数据里复制一段不太合适的代码。

## 5. ID重复Bug

```python
# 错误的ID生成
def add(self):
    todo = {'id': len(self.todos) + 1, ...}

# 删除后添加，ID会重复
```
