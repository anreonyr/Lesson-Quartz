# 项目规划

在开始编程之前，先用自然语言描述你的需求。

## 需求文档

- 程序类型：桌面图形界面应用程序
- 编程语言：Python
- 图形库：Tkinter（内置，无需安装）
- 数据存储：JSON文件
- 功能列表：
  - 添加：用户输入内容，点击按钮添加到列表
  - 查看：显示所有未完成和已完成的待办
  - 完成：点击待办事项标记为已完成
  - 删除：选中待办后点击删除按钮

## 模块划分

1. `todo_gui.py` - 图形界面主程序
2. `todo.py` - 待办事项数据模型
3. `storage.py` - 数据持久化（JSON读写）

## 数据格式

```json
{
  "todos": [
    {
      "id": 1,
      "content": "学习AI编程",
      "completed": false,
      "created_at": "2026-03-08T10:00:00"
    }
  ]
}
```

## 开发流程

1. 明确需求
2. 设计架构
3. 分步实现
4. 组合调试
5. 测试和修复
6. 代码审阅
