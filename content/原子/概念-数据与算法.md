# 数据与算法

程序 = 数据 + 算法

## 数据

数据是程序处理的信息：

- 数字（整数、浮点数）
- 文本（字符串）
- 布尔值（真/假）
- 复杂结构（数组、对象、列表）

## 算法

算法是解决问题的步骤：

- 顺序执行
- 条件判断（if/else）
- 循环（for/while）
- 函数/方法

## 数据与算法的关系

数据是"原料"，算法是"加工方法"。

```
输入数据 → 算法处理 → 输出数据
```

## 与 AI 编程的关系

理解数据与算法有助于：

- 设计清晰的数据结构
- 规划算法的处理步骤
- 与 AI 沟通时更准确地描述需求
