# 什么是库

库（Library）是他人已经写好的功能代码，可以直接拿来用。

## 库的特点

- 预先编写好的代码模块
- 提供特定功能
- 可以重复使用
- 通常经过测试和维护

## 常见类型

| 类型 | 示例 |
|------|------|
| Python库 | requests, pandas, numpy |
| JS包 | react, vue, lodash |
| 框架 | Django, Flask, Express |

## 为什么使用库

1. 不重复造轮子
2. 提高效率
3. 质量有保证
4. 社区支持
