# Prompt基础

Prompt是你向AI发送的消息，包含了你希望AI完成的任务描述。

## 好的Prompt特点

- **清晰**：让AI知道你要做什么
- **具体**：提供足够的细节
- **结构化**：有逻辑地组织信息

## 基本结构

```
[角色设定（可选）] + [任务描述] + [具体要求] + [格式要求]
```

## 示例

> 你是一个Python程序员。请写一个函数，计算列表中的平均值。要求：
> - 输入是一个数字列表
> - 返回平均值
> - 如果列表为空，返回None

## 为什么Prompt重要

- 好的prompt带来好的回答
- 模糊的prompt导致无用的答案
- 需要不断迭代优化
