# Agent工具

Agent工具是理解任务、调度工具、生成回答的AI助手。

## 核心能力

- 理解自然语言任务
- 调度各种工具完成任务
- 读写本地文件
- 执行终端命令
- 搜索网页信息

## 常见Agent工具

| 工具 | 特点 |
|------|------|
| Claude Code | 深度理解代码库，安全优先 |
| Cursor | 与VS Code深度集成 |
| GitHub Copilot | 集成在IDE中的代码补全 |
| OpenCode | 多模型支持，双模式 |

## 与对话模式的区别

| 特性 | 对话模式 | Agent模式 |
|------|----------|-----------|
| 操作范围 | 回答问题 | 执行操作 |
| 文件操作 | 只能看 | 可读写 |
| 命令执行 | 不能 | 可以 |
| 自动化程度 | 低 | 高 |
