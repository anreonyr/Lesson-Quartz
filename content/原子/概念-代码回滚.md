# 代码回滚

当AI改坏了代码时，如何回滚？

## 回滚方法

### 1. 放弃当前修改

```bash
# 放弃工作区修改
git checkout .

# 放弃暂存区修改
git reset HEAD
```

### 2. 回退到某个提交

```bash
# 回退到上一个提交
git reset --hard HEAD~1

# 回退到指定提交
git reset --hard <commit-id>
```

### 3. 用Lazygit回退

1. 运行 `lazygit`
2. 在提交历史中找到正确的版本
3. 按 `g` 签出到那个版本

## 注意事项

- `--hard` 会丢失未提交的修改
- 建议先 `git commit` 保存当前状态
- 回退前确认要回到哪个版本

## AI编程建议

- 每次让AI大改之前先提交当前版本
- 用Lazygit查看AI具体改了什么
- 确认修改符合预期后再合并
